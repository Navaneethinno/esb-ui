## 🔍 Implementation Details

### Auto-Generation Algorithm

```python
def _next_mapping_id(self, inbound_adapter_id, outbound_adapter_id):
    """Generate IB-{OUTBOUND_NAME}-{INBOUND_NAME}-{SEQUENCE} mapping ID."""
    
    # Step 1: Clean adapter name (remove non-alphanumeric, truncate to 12 chars)
    def _clean(name):
        slug = re.sub(r'[^A-Za-z0-9]+', '', str(name or '')).upper()
        return slug[:12] or 'UNKNOWN'
    
    # Step 2: Fetch names from database
    ib_name = _clean(fetch_inbound_adapter_name(inbound_adapter_id))
    ob_name = _clean(fetch_outbound_adapter_name(outbound_adapter_id))
    
    # Step 3: Build prefix
    prefix = f"IB-{ob_name}-{ib_name}"
    
    # Step 4: Get sequence number (count existing + 1)
    seq = count_existing_mappings(inbound_adapter_id, outbound_adapter_id) + 1
    
    # Step 5: Return formatted ID
    return f"{prefix}-{seq:04d}"
```

### A. Database Schema
```sql
CREATE TABLE adapter_link_mapping (
    id SERIAL PRIMARY KEY,
    mapping_id VARCHAR(100),  -- Auto-generated: IB-XXX-YYY-####
    mapping_name TEXT,        -- Optional (legacy support)
    inbound_adapter_id VARCHAR(100) NOT NULL,
    outbound_adapter_id VARCHAR(100) NOT NULL,
    inbound_request_name VARCHAR(150),
    outbound_request_name VARCHAR(150),
    request_mappings JSONB NOT NULL DEFAULT '{}'::jsonb,
    response_mappings JSONB NOT NULL DEFAULT '{}'::jsonb,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    UNIQUE (inbound_adapter_id, outbound_adapter_id)
);
```
